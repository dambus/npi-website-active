# Portfolio Export Plan

## Future Feature

Users should later be able to select multiple references and generate a PDF portfolio package.

## Phase 4 Scope Boundary

- This feature is not implemented in Phase 4.
- It is documented now so it stays visible during future references module work.

## Likely Product Requirements

- selection state in the references listing
- export action or button
- curated PDF output with a clean corporate layout
- reference summary content suitable for print/share workflows

## Likely Technical Directions

- server-side PDF generation for consistent output
- client-side selection state with export payload submission
- reusable portfolio template that draws from the same project data model used by the public site
