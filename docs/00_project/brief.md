# Brief

## Project

NPI website rebuild for a concise corporate presentation site with a future Supabase-backed admin area.

## Locked V1 Public Scope

- Home
- About
- Services
- References
- Contact

## V1 Priorities

- establish a credible, premium industrial web presence
- communicate services and delivery model clearly
- make references/projects the main proof-driven content module
- keep public content concise and easy to scan

## CMS Boundaries For This Stage

- schema, storage, and admin rules are planned in this phase
- final UI implementation does not start yet
- real CRUD and real auth do not start yet
- news stays secondary to references in both scope and effort

## Outcome Of This Documentation Phase

This phase locks the sitemap, page content scope, homepage structure, design direction, references UX, schema draft, and execution plan so the next phase can move into implementation with fewer open questions.
